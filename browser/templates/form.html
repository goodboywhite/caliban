Select a file that you would like use to see how Caliban works!

The .trk files allow you to annotate RAW264 cells across time-lapse frames while preserving relationship information.

The .npz files allow you to annotate different types of data.
test.npz is an example of organoid annotation across z-stack frames.
test2.npz is an example of untracked HeLa cytoplasm annotations in a time-lapse movie (5 frames).
test3.npz is an example of zslices from a chicken embryo.

The .npz files are recommended for trying out Caliban's annotation correction features, while the .trk files are recommended as an example of how the tool can be adapted to particular use cases.

Caliban currently supports whole-label editing options and some basic pixel-editing tools. Advanced pixel-editing tools will be coming soon.

These files will not be overwritten with any changes, so feel free to try any and all of Caliban's functionality. Refreshing the page with a file open will reload the unmodified file.
